## Usage

1. Launch the application.
2. Use the "Browse" button to select a folder to search for duplicates.
3. Click "Search for Duplicates" to start the search process.
4. Once the search is complete, review the results in the tree view.
5. Select duplicate groups or individual files for deletion.
6. Use the "Delete Selected" or "Delete All Except First" buttons to remove duplicates.
7. Use the preview pane to view file contents and details.
8. Open files or their containing folders directly from the application.

## Customization

The application offers multiple themes:
- Dark (default)
- Light
- Solarized
- Nord

To change the theme, use the dropdown menu in the top-left corner of the application.
